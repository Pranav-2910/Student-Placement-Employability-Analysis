# 🎓 Student Placement & Employability Analytics Dashboard

## 📌 Overview

The **Student Placement & Employability Analytics Dashboard** is a data analytics project developed using **Python, SQL, and Power BI** to analyze student placement trends and employability factors. The project transforms raw student data into meaningful insights through data cleaning, feature engineering, and interactive visualizations.

This dashboard enables educational institutions, placement cells, and career development teams to identify placement patterns, assess employability readiness, and make data-driven decisions.

---

## 🚀 Project Objectives

* Analyze placement performance across academic branches and degrees.
* Measure placement percentages and employability metrics.
* Evaluate the impact of skills and experience on placement outcomes.
* Compare placement trends based on gender and age groups.
* Develop an interactive dashboard for decision-making.

## 🔍 Key Insights

* CSE students recorded the highest placement count among all branches.
* IT students closely followed CSE in placement performance.
* MCA students achieved the highest placement numbers among degree programs.
* Students with higher skill scores demonstrated better placement outcomes.
* Internships, projects, and certifications significantly improved employability.
* Experience Score showed a strong positive relationship with placement success.
* Placement trends varied across age groups and academic backgrounds.
